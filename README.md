# Gabe's Forest Plotter

A simple tool for making clean forest plots for manuscripts, posters, and presentations.

<a href="https://gabrielbonassi77.github.io/Gabe-Forest-Plotter/" target="_blank" rel="noopener noreferrer">https://gabrielbonassi77.github.io/Gabe-Forest-Plotter/</a>

![Example forest plot](https://github.com/GabrielBonassi77/Gabe-Forest-Plotter/blob/main/FigureExample.png "Example forest plot")

## How to Use

1. Open the web app.
2. Upload a CSV or Excel file, or edit the example data already in the table.
3. Adjust the axis, tick marks, spacing, font size, or marker size if needed.
4. Download your figure as PDF, PNG, TIFF, or SVG.

## Data Format

Your file should have these columns:

| Outcome | Mid point | Low CI | High CI |
| --- | ---: | ---: | ---: |
| Example outcome | 1.31 | 1.18 | 1.45 |

The app also accepts similar column names like `Mid`, `Lo`, and `Hi`.
